Renamer is an apllication for renaming multiple files at once. It is meant to be used with sets of related files such as music or photos that should all be named in a similar way. It works by manipulating the original filename with different modifing techniques.

String Modifiers

  • Prefix - Add text to begining of filename
  • Suffix - Add text to end of filename
  • Counter (WIP) - Add incrementing number to list of files
  • Substring (WIP) - Reduce whole string to a subset of itself
  • Replace (WIP) - Replace one string with another

Code

The individual classes have not been pinned down yet but I have got one for holding the list and modifing it and a main execution class.

The code will utilize an object to hold the filenames in a string array field. They will never be dirrectly modified. They will be modified when needed by a list of actions and values that parse the names through the objects methods. The actions are stored in a strin array that uses delimiters to contain the different values. For example, the replace's entry would look like this 4mispeled/spelled. The first character 4 is the method or action's ID. The mispeled is the text to be replaced and the spelled is what will replace it. The forward slash is used as a delimiter since it is not allowed in filesystems.
